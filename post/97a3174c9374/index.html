官网地址

安装elasticsearch

  1. 下载elasticsearch8

  2. 为 Elasticsearch 和 Kibana 创建新的 docker 网络

  3. 运行elasticsearch
    容器启动后会生成一个有限期30分钟的token，用于后续的kibana的登录，如果没有跳至第6步

  4. 开启新窗口设置密码或重置密码

  5. 检查安装
    打开https://127.0.0.1:9200，输入用户名(`elastic`)和密码登录，成功后出现如下信息表示安装并登录成功

  6. 生成token
